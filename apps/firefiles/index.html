FireFiles Features

summarise what your product is all about
Attractive Layout

Its a super File Manager which provides all the key features of basic file management including Cutting, Copying, Pasting, Deleting, Renaming, Compression and Extraction of files. All these come in an attractive layout..

Smart Library File Manager

FireFiles is developed with the ability to categorize all your files into Downloads from internet, Bluetooth from nearby devices, Images, Pictures, Videos, Movies, Audio, Music, Documents (pdf, xls, ppt, etc), Archives (zip, rar, etc) and APKs.

Storage File Manager

With FireFiles, you can access all files and folders from different types of storages like EMMC, MMC Cards, SD Cards, Internal Storage, External Storage, Phone Memory, USB Storage, Flash Drives, USB OTG, Pen Drives (Nexus and Non-Nexus devices).

Apps and Process Manager

Monitor and Kill your memory consuming applications with FireFiles. Quickly and easily unistall unwanted programs to free up storage space whic is very valuable. FireFiles comes with support for FAT file systems too.

Network Manager/File Transfer

Transfer files with FTP Server from your device to PC with FireFiles using Network File Transfer Protocol. All you have to do is to open the IP address in any browser and all your files on the phone are easily accessible by the PC

Root File Manager

If your device is rooted, you would definitely love this unique operation we added for advanced users to explore, edit, copy, paste and delete files in the root partition of the phone storage for development purposes. YOUR DEVICE NEEDS TO BE ROOTED TO USE THIS FEATURE!

Internal Document Editor

Now with FireFiles, you can easily edit files on the go. FireFiles support every text type such as HTML, XHTML, TXT, etc. You can also edit your files even if your deviced is not rooted. It is very simple and handy!

Size

FireFiles is one of the first File Managers that supports Android TV, supports RTL languages like: Arabic, Hebrew, etc; all done natively! It is among the rarest File Manager apps that contains this much features and still maintain a relatively small size that is less than 5MB (3MB)...
